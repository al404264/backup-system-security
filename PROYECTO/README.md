# Sistema de Copias de Seguridad con Encriptación

Este proyecto es un sistema automatizado de copias de seguridad utilizando scripts en Bash. Las copias de seguridad son incrementales y se cifran utilizando AES-256. Los scripts también gestionan el espacio de almacenamiento y envían notificaciones cuando se alcanza el 90% de uso del almacenamiento.
